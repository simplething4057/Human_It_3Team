# GitHub 코드 다운로드 가이드

GitHub에서 프로젝트 코드를 받아오는 방법은 크게 두 가지가 있습니다.

## 1. Git을 사용한 방법 (권장)
현재 시스템에 `git`이 설치되어 있으므로, 다음 명령어를 사용하여 코드를 바로 받아올 수 있습니다.

```powershell
git clone https://github.com/simplething4057/Human_It_3Team.git
```

## 2. GitHub CLI (gh)를 사용하는 방법
사용자께서 요청하신 `gh repo clone` 명령어는 **GitHub CLI**가 설치되어 있어야 작동합니다. 현재 시스템에는 `gh`가 설치되어 있지 않아 오류가 발생했습니다. 만약 GitHub CLI를 사용하고 싶으시다면 먼저 설치가 필요합니다.

### 명령어:
```powershell
gh repo clone simplething4057/Human_It_3Team
```

---

> [!NOTE]
> 현재 `gh` 명령어가 인식되지 않아, 제가 대신 **Git**을 사용하여 코드를 받아오도록 시도하겠습니다.
